 Branch Git 
=============

- 產生 Branch
  
    > `git branch [brancName]`
	
- 列出全部 Branch (local branch)
	
    > `git branch --list`
    > `git branch -a`

- 切換不同 Branch
  
    > `git checkout [branchName]`

- 產生，且切換 Branch
  
    > `git checkout -b [branchName]`
	
- 刪除 Branch
	
    > `git branch -d [branchName]`


Git pull 
=========

    > `git pull [remote]`
